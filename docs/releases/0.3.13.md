# Walk 0.3.13

- Bundles Everything directly into Walk so fast indexed file search works without any separate install.
- Adds wildcard file search such as `*.pdf` and removes the old fixed result limit so large result sets can scroll.
- Adds fuzzy file search and full-path reranking so typo-style queries like `smthng` can still surface `something.mp4`.
- Adds an in-search animated loading indicator so long-running searches are easier to understand.
